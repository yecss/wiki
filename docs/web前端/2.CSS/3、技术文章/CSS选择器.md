## 选择器的分类

### 标签选择器

```css
h1 {color:blue;}
```



### 类选择器

```css
.important {color:red;}
```



### ID选择器

```css
#intro {font-weight:bold;}
```



### 属性选择器

```css
a[href] {color:red;}
```



### 后代选择器（可跨多代）

```css
h1 em {color:red;}
```



### 子元素选择器

```css
h1 > strong {color:red;}
```



### 通配符选择器

```css
* {color:red;}
```
