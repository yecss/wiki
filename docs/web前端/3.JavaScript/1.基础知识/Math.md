## 数学运算Math

math是一个工具类不用创建对象，它里面封装了和数学运算相关的属性和方法。

### 属性

```JavaScript
console.log(Math.PI) //打印π
```

### 方法

`abs()`

计算一个数的绝对值

`ceil()`

可以对一个数进行向上取整，小数位只要有就自动进一

`floor()`

可以对一个数进行向下取整，小数位只要有就自动进一

`round()`

可以对一个数四舍五入取整

`random()`

生成一个0-1之间的随机数

- **Math.random()*10**  整体乘以10
- **Math.round(Math.random()*x))** 生成一个0-x之间的整数

`max()和min()`

获取多个数中的最大值和最小值

`pow(x,y)`

返回x的y次幂

`sqrt()`

对一个数进行开方
