### 标识符1

C语言中的标识符只能由字母、数字和下划线三种字符组成，且第一个字符必须是字母或下划线。

标识符是用户编程时使用的名字，比如指定某个东西、人，都要用到它，他或她的名字。在电脑语言中，对于变量，常量，函数，语句块也有名字，统统称之为标识符。

### Fibonacci数列

### return 0

return 0 是函数的返回值,

函数中要写明return 0,表示返回。

因为函数(除void)在遇到return时才能返回。

一定要有,代表程序结束。

**用途**：return 0用来告知程序的结束。

### 数组

```c

```
